Filmorate — это бэкенд-платформа социальной сети, созданная для людей, увлеченных миром кино. Сервис помогает пользователям открывать новые фильмы через рекомендации единомышленников, делиться впечатлениями и находить друзей с похожими вкусами.
Это групповой проект, разработанный командой:  
[vBoltunov](https://github.com/vBoltunov), [TireNik](https://github.com/TireNik), [AlionaKulikova](https://github.com/AlionaKulikova), [marloq1](https://github.com/marloq1)

## Ключевая ценность проекта

Этот проект демонстрирует реализацию комплексной системы управления данными и социальными взаимодействиями, включая:

* **Персонализированные рекомендации** на основе анализа предпочтений пользователя и его друзей
* **Социальный функционал** — друзья, общие интересы, лента активности
* **Гибкий поиск и фильтрацию** контента по различным критериям
* **Расширенную работу с контентом:** рейтинги, отзывы, группировка по режиссерам и жанрам

## Что было реализовано

- **RESTful API** с четкой структурой и бизнес-логикой
- **Система рекомендаций**, использующая анализ пользовательских данных
- **Механизмы социального взаимодействия** (друзья, лента событий)
- **Поисковая система** по фильмам
- **Управление разнородными сущностями** (фильмы, пользователи, отзывы, режиссеры)
- **Реализация популярных сценариев:** "Топ-N фильмов", "Общие с друзьями фильмы"

Проект выполнен с акцентом на архитектурную чистоту, производительность и расширяемость, демонстрируя подходы к построению современных веб-сервисов.

## Запуск проекта
1. Клонировать репозиторий  
   ```bash
   git clone https://github.com/vBoltunov/filmorate.git
   cd filmorate
   ```

2. Собрать проект  
   ```bash
   mvn clean install
   ```

3. Запустить  
   ```bash
   mvn spring-boot:run
   ```
